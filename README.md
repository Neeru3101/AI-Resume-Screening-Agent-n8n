# AI-Resume-Screening-Agent-n8n
AI-powered Resume Screening System built using n8n, Groq/OpenAI LLM and workflow automation.

🤖 AI Resume Screening Agent (n8n + LLM)
📌 Project Overview
This project is an AI-powered Resume Screening System built using n8n workflow automation and LLM integration.
The system automates resume parsing and candidate evaluation using AI-based analysis.

🚀 Features
📄 Resume Upload via Form
📂 PDF Parsing & Text Extraction
🧠 AI-based Personal Information Extraction
💼 Professional Skills Analysis
📊 Structured Candidate Data Output
⚡ Automated Workflow using n8n

🛠 Tech Stack
n8n (Workflow Automation)
Groq / OpenAI LLM
Google Drive
PDF Extract Node
AI Agent Logic
Webhook / Form Trigger

🧠 Workflow Architecture
User submits resume via form
File uploaded to Google Drive
Resume text extracted
LLM analyzes:
Personal Information
Professional Information
Data merged and structured

Final output generated
